Api documentation

Send a new logging request

  • Endpoint: /api/log
  • Method: POST
  • Paramters required: information, api_key
  • information: a string containing the information you want to log
  • api_key: your supplied api key

Retrieve logging information

  • Endpoint: /api/log
  • Method: GET
  • Parameters required: api_key
  • Optional parameters: hour, day, week
  • api_key: your supplied api key
  • hour: get all log information within the latest hour
  • day: get all log information within the latest day
  • week: get all log information within the latest week
